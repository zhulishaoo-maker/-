# Prompt 可编辑槽位设计

## 目标

将 Prompt Composer 中除任务类型外的灰色下拉槽位升级为可编辑组合框。运营既能使用推荐值，也能直接输入活动主题、品类、权益、压板名称、风格与尺寸，并即时同步到最终 Prompt 和 `GenerationBrief`。

## 交互

- 点击槽位后聚焦文本并打开推荐列表。
- 输入任意文字时保留当前内容，并按文字过滤推荐项。
- 点击推荐项或按 Enter 确认；按 Escape 取消本次修改。
- 失焦时保存非空内容；空内容恢复进入编辑前的有效值。
- 使用上下方向键移动高亮项，当前高亮项使用清晰的视觉反馈。
- 下拉箭头用于表达仍有推荐值，不限制自由输入。
- “开屏”等任务类型继续由下方任务卡切换，不进入本次组合框范围。

## 组件边界

新增独立 `EditableSlot` 组件，负责输入状态、推荐过滤、键盘导航、失焦提交和无障碍属性；`PromptComposer` 只负责句式编排及更新 `ComposerState`。推荐项为本地确定性配置，自定义值不写回推荐库。

## 无障碍与异常处理

- 输入框使用 `combobox` 语义，并关联 `listbox` 与当前高亮项。
- 推荐项可使用鼠标和键盘选择。
- 空字符串不允许覆盖现有 Brief 数据。
- 输入过长时通过字段级最大长度限制控制，避免破坏句式布局。

## 验收

- 七个灰色槽位均可自由输入并选择推荐项。
- 自定义值出现在自然语言 Prompt 和生成结果页。
- Enter、Escape、上下方向键和点击外部均符合定义。
- 桌面端不产生横向溢出；单测、类型检查、构建和浏览器控制台通过。
